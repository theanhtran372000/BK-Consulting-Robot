# BK Consulting Robot - Frontend
Source code cho **frontend server** (*Khối quản trị*) trong hệ thống BKBot.

## 1. Yêu cầu hệ thống
- Thiết lập môi trường
```
npm install
```

## 2. Chạy chương trình
- Build chương trình
```
npm run build
```
- Chạy chương trình đã build
```
serve -s build
```
